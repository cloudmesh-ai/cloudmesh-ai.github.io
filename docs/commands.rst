Commands
========

Cloudmesh AI provides a variety of CLI commands to manage your environment.

Core Commands
-------------

* ``cmc version``: Display the current version of the tool.
* ``cmc config``: Manage configuration settings.
* ``cmc plugins``: Manage installed plugins.
* ``cmc doctor``: Run system diagnostics.

System Information
------------------

* ``cmc sys info``: Display detailed system and environment information.

Utility Commands
----------------

* ``cmc time``: Display current time and synchronization status.
* ``cmc logs``: View system and application logs.
