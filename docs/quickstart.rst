Quickstart
===========

This guide will help you get started with Cloudmesh AI.

Installation
------------

To install the core components, use pip:

.. code-block:: bash

   pip install cloudmesh-ai-cmc cloudmesh-ai-common

Basic Usage
-----------

Once installed, you can use the `cmc` command to manage your AI environment.

.. code-block:: bash

   cmc --help
