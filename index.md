Welcome to the **Cloudmesh AI** ecosystem. We provide tools for seamless AI orchestration on HPC environments, including NVIDIA DGX Stations and A100/Blackwell clusters.

## Core Projects

* **[AI Shell](shell/index.qmd)**: A specialized CLI for managing LLMs and RAG workflows.
* **[Tracker Native]**: Mobile-first avian monitoring and triangulation.
* **[Benchmarks]**: Comparative analysis of YOLO models and RAG performance.

> [!TIP]
> Use `pip install cloudmesh-cma` to get started with the core shell.
